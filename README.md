# SleepFM Latent Space Exploration

This project explores pretrained SleepFM embeddings using unsupervised learning techniques. 

Goals:
- Load pretrained embeddings from SleepFM
- Explore latent structure using dimensionality reduction
- Visualize classical sleep stages (Wake, N1, N2, N3, REM)
- Identify potential transitional structures in embedding space

Dataset:
Sleep-EDF (PhysioNet)

Tech Stack:
Python, PyTorch, Scikit-learn, UMAP, Kaggle
